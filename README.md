## 这个工具是干嘛的？

**团子转换器** 是一个 **极度傻瓜化** 的 Markdown → Word（`.docx`）桌面工具。

它解决的是一个非常具体、但被长期忽视的问题：

> 👉 **如何把 AI（ChatGPT / Claude / DeepSeek 等）的回答，  
> 快速、干净、可控地变成 Word 文件？**

如果你经常遇到这些情况，这个工具就是为你做的：

- AI 写了一大段内容  
  👉 复制到 Word 里 **格式全乱**
- 列表、标题、代码块、引用  
  👉 **要手动调半天**
- 在线 Markdown 转 Word 网站  
  👉 **效果不全 / 操作复杂 / 限制多 / 要收费**
- 自己用 Pandoc  
  👉 **命令行太劝退**

**团子转换器 = 把 Pandoc 的强大能力，封装成“点一下就能用”的 GUI。**

---

## 它适合谁？

- 🧠 **AI 重度用户**
- ✍️ 用 AI 写：
  - 文章 / 报告 / 作业
  - 技术文档 / 方案 / 说明书
- 📄 最终交付物必须是 **Word**
- ❌ 不想学 Markdown / Pandoc / 命令行
- ❌ 不想被在线工具割韭菜

## 核心特点

- ✅ **专为 AI 输出设计**
  - 直接粘贴 AI 回答即可
- ✅ **一键导出 Word（.docx）**
- ✅ 支持：
  - 标题 / 列表 / 引用
  - 代码块
  - Markdown 基础语法
- ✅ 内置多套 Word 模板
  - 字体
  - 加粗规则
  - 页码 / 正文样式
- ✅ 完全本地运行
  - **不联网**
  - **不上传内容**
  - **不收集数据**

---

## 为什么不用在线转换网站？

说人话版原因👇

- ❌ **效果不可控**
  - 列表、标题经常乱
- ❌ **操作反人类**
  - 各种选项、步骤一堆
- ❌ **限制多**
  - 文件大小
  - 转换次数
- ❌ **迟早收费**
- ❌ **内容要上传**
  - 隐私不安心

而团子转换器：

- ✔ 本地
- ✔ 免费
- ✔ 点一下就完事

## 下载与使用（推荐）

### 🚀 给普通用户（强烈推荐）

1. 打开本仓库的 **Releases** 页面
2. 下载 `Markdown转Word.exe`
3. 双击运行即可

> ✅ Release 版本已 **内置 Pandoc**  
> ✅ **无需安装 Python / Pandoc / 任何依赖**

---

## 基本使用流程

1. 从 AI 工具中复制回答
2. 打开团子转换器
3. 粘贴 / 拖拽 Markdown 内容
4. 选择模板
5. 导出 `.docx`

**结束。**
